# ACM - AWS Certificate Manager

## 証明書の種類

### DV / OV / EV
- **DV**：ドメインの存在確認 → ACM
- **OV**：組織の法的存在確認 → 外部認証局
- **EV**：組織の物理的存在確認（金融など）→ 外部認証局

### パブリック証明書
- 無料
- 自動更新
- 検証要
- 公開サーバー向け

### プライベート証明書
- 有料（Private CA + 証明書単位）
- 有効期限を設定
- 検証不要
- 社内システム向け

## ALBでのSSLターミネーション
- HTTPS通信をALBで終端
- 内部通信はHTTP
- 証明書管理が一元化
- サーバーの復号処理が不要 → パフォーマンス向上
