# DirectConnect
## 接続
### 専用接続
- 単一のアカウントに関連付け
- 1、10、100Gbps

### ホスト接続
- パートナーがユーザーに割り振る
- 50、100、200、300、400、500Mbps

## 回復性レベル
# 最大回復性
- ローケーションとデバイスの両方を冗長

# 高い回復性
- ロケーションのみ冗長

# 開発とテスト
- デバイスのみ冗長

## 仮想インターフェイス
### プライベート
- VPCにアクセスする時
- データ転送を行うawsアカウントに課金

### パブリック
- パブリックIPアドレスでawsパブリックサービスにアクセスする時
- リソース所有者に課金

### トランジット
- トランジットゲートウェイにトラフィックを転送するとき

## Direct Connect Gateway
- プライベート仮想インターフェイスをDirect Connect Gatewayに接続して異なるリージョンにあるVPC含むにアタッチされた仮想プライベートゲートウェイを10まで関連付けれる
- それ以上はトランジット仮想インターフェイスを使う

## LAG
- 専用線を束ねて一つの接続にする
